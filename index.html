O que são Requisitos

Requisitos definem o que um sistema deve fazer e sob quais restrições. O que um sistema deve fazer, as suas funcionalidades, são chamados
de Requisitos Funcionais. Já os requisitos relacionados com a segunda parte são chamados de Requisitos Não-Funcionais.
